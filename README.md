# system_usage_stats
Scripts to ease calculating peak/average CPU, Load and Disk usage

# Prerequisites

sysstat
MongoDB

# Usage

./collect_stats.sh <TEST_TIME> <TEST_NAME>

# Output

Peak Load:      0.34
Peak CPU Usage: 1.1
Avg CPU Usage:  0.85
Peak Queries:   42.0
Avg Queries:    22.2222222222
Peak Disk:      11.36
Avg Disk:       5.815
